# 2D Game

A simple physics engine and game created in Java for my independent study for Seabury Hall.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Download [Intellij Idea](https://www.jetbrains.com/idea/download)

### Installing

A step by step series of examples that tell you have to get a development env running

* Install Intellij Idea
* In the home screen click `Check out from Version Control` and click `GitHub`
* in the Git Repository URL, paste in `https://github.com/rmccrystal/2dgame.git`
* Select the directory you want to use to store the project
* Click `Clone`
* Click yes to open the project

## Building

* In the Intellij project, click the green triangle on the top right or go to `Run` and click `Run 'Main'`
